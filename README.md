# Qlever Docs

These are the docs for the graph database [QLever](https://github.com/ad-freiburg/qlever) and the accompanying tooling like [QLever UI](https://github.com/qlever-dev/qlever-ui) and [qlever-control](https://github.com/qlever-dev/qlever-control). View the documentation at <https://docs.qlever.dev>.

## Developing locally

1. Clone this repo
2. Setup a virtual environment with `uv venv` or `python3 -m venv .venv` and activate it `. .venv/bin/activate`
3. Install the dependencies with `uv sync` or `pip install -e .`
4. Run the development server `mkdocs serve`
